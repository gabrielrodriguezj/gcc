# gcc
Gabriel's C Compiler: An educational C compiler built for learning and fun.

This compiler was developed primarily from Writing a C Compiler: Build a Real Programming Language from Scratch by Nora Sandler, with additional references consulted throughout the project.

## References
* Sandler, N. (2024). Writing a C compiler: Build a real programming language from scratch. No Starch Press.
* Nystrom, R. (2021). Crafting interpreters. Genever Benning.
