# gcc
Gabriel's C Compiler. A C compiler for recreational and educational purposes.

This compiler was written primarily following the book "Writing a C Compiler: Build a Real Programming Language from Scratch" by Nora Sandler. Other sources were consulted.

## References
* Sandler, N. (2024). Writing a C compiler: Build a real programming language from scratch. No Starch Press.
* Nystrom, R. (2021). Crafting interpreters. Genever Benning.
